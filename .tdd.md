#### Implementation Notes (Phase 8)
* **Loader Init Flow**: `main` で `LoadCards → LoadNations → LoadEnemies → LoadBosses` の順に呼び出し、得たマップを各 Manager 生成時に DI する。
* **Manager Refactors**:
  * `CardManager` -> `NewCardManager(data map[string]*entity.Card)` に変更、`CreateCard(id)` でテンプレ複製。
  * `AllianceManager` -> `NewAllianceManager(nations map[string]*entity.Nation)` で初期関係値を設定。
  * `CombatManager` -> `NewCombatManager(enemies map[string]*entity.Enemy, bosses map[string]*entity.Boss)` に変更し、`AddEnemyByID` などを提供。
* **MapGrid**: Wild / Boss ポイントに `EnemyID` / `BossID` を保持し、クリック時に CombatManager へ ID 渡し。
* **Hot-Reload (Nice to have)**: `loader.Watch()` でファイル更新を検知し、各 Manager の `Refresh` メソッドに反映。
